Ejercicios de Python pasados a JavaScript

Los ejercicios planteados, fueron diseñados para ser resueltos en la sintaxis de Python, pero ahora van a ser trasladados a JavaScript, siendo el mismo funcionamiento de estos mismos, pero con un ligero cambio. Con la explicación de estos, ya no serán tan complicados de entender.

¿Qué herramientas usamos para crear esta página?

Todo lo que usamos fueron 3 programas para completar la página y con ellos realizamos los ejercicios en la misma, los cuales son:

HTML

(HyperText Markup Language) es un lenguaje de marcado utilizado para crear y estructurar contenido en la web, como texto, imágenes, videos y enlaces, utilizando etiquetas predefinidas para definir su apariencia y función.

CSS

(Cascading Style Sheets) es un lenguaje de diseño utilizado para dar estilo y formato a documentos HTML, permitiendo crear diseños atractivos y visualmente agradables.

JavaScript

JavaScript es un lenguaje de programación utilizado para crear interacciones dinámicas en páginas web, permitiendo a los desarrolladores crear efectos visuales y animaciones, validar formularios, entre otras funcionalidades.

Cómo hicimos el esquema, las animaciones y los colores de la página

Esta pagina la hicimos por medio de programas como HTML,CSS y JavaScript, agregando tambien un ligero complemento con el Framework Boostrap, nos enfocamos en utilizar diferentes span y div, para guardar en un cuadro cada elemento que apareciera en la pagina, tambien nos guiamos de otras paginas y videos para llegar al resultado final, para los ejercicios usamos algunos ya hechos en Python pero los cambiamos para que se adaptaran a la sintaxis de Javascrip.
